```bash
ujust update
rpm-ostree install alacritty niri xwayland-satellite
```

# CLI install

```bash
brew install atuin bat btop bzip2 cava chafa chezmoi distrobox eza fd fzf gcc lazygit neovim pipx podman-compose sevenzip task tree-sitter tree-sitter-cli yazi
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

# Set up containers

```bash
sh ~/.local/share/chezmoi/scripts/init_containers.sh
```

# Install Noctalia

